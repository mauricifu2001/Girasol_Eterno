# Audio de fondo

Pon aqui el archivo de audio local que quieras reproducir cuando entre Valentina.

## Recomendacion

- Usa un archivo corto o liviano en formato `.mp3` o `.ogg`.
- El nombre configurado por defecto es `valentina-theme.mp3`.
- Si usas otro nombre, cambia `portal.soundtrack.src` en `config.js`.

## Importante

- No uses un link de YouTube como pista de fondo directa.
- Para esta web, lo correcto es usar un archivo de audio que tengas guardado localmente y que puedas usar.
- Si vuelves a la pantalla de acceso y se reactiva la camara, la musica se detiene.
