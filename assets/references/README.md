# Fotos de referencia

Pon aqui las fotos de las personas autorizadas.

## Recomendaciones

- Usa al menos 2 imagenes por persona.
- Mejor si estan de frente y con buena iluminacion.
- Evita filtros, gafas oscuras o fotos muy lejanas.
- Cambia los nombres de archivo en `config.js` si no quieres usar los que vienen por defecto.

- `persona-1-b.jpeg`
